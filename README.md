<div id="architecture" style="border:4px solid #050505;background:#FFF5E6;color:#050505;box-shadow:10px 10px 0 #050505;padding:18px;margin:16px 0;">
<div style="display:flex;align-items:center;gap:10px;margin-bottom:12px;">
<div style="padding:6px 10px;border:3px solid #050505;background:#050505;color:#FF66C4;font-family:'Space Mono','Courier New',monospace;font-weight:900;">03 / Architecture</div>
<div style="height:3px;flex:1;background:#050505;"></div>
</div>
<p style="margin:6px 0;font-size:15px;line-height:1.65;color:#050505;">This repo is the <strong>frontend</strong> (React + Vite). AI requests are routed through a <strong>backend proxy</strong> so secrets never hit the browser and the API surface stays stable.</p>

<div style="margin-top:12px;padding:12px;border:3px solid #050505;background:#FFDE59;box-shadow:6px 6px 0 #050505;font-family:'Space Mono','Courier New',monospace;font-size:13px;line-height:1.6;color:#050505;">
<div style="font-weight:900;">REQUEST FLOW (SEQUENCE)</div>
<pre style="margin:8px 0 0 0;font-family:'Space Mono','Courier New',monospace;font-size:13px;line-height:1.6;color:#050505;">Visitor UI
	│
	├─(1) userMessage + portfolio context
	│
	├─(2) POST https://gemini-backend-portfolio.vercel.app/api/gemini
	│        body: { userMessage, context }
	│
	├─(3) Proxy calls Gemini (server-side credentials)
	│
	└─(4) UI receives { text, sources? }</pre>
</div>

<table style="width:100%;border-collapse:separate;border-spacing:10px;margin-top:10px;">
<tr>
<td style="width:50%;vertical-align:top;padding:12px;border:3px solid #050505;background:#FFFFFF;box-shadow:6px 6px 0 #050505;color:#050505;line-height:1.6;">
<strong>Frontend</strong><br/>
Calls the proxy from <code>services/geminiService.ts</code> and sends:
<ul style="margin:8px 0 0 18px;line-height:1.6;color:#050505;">
<li><code>userMessage</code> (visitor input)</li>
<li><code>context</code> (curated portfolio + tone)</li>
</ul>
</td>
<td style="width:50%;vertical-align:top;padding:12px;border:3px solid #050505;background:#E0F2FE;box-shadow:6px 6px 0 #050505;color:#050505;line-height:1.6;">
<strong>Backend Proxy</strong><br/>
Owns secrets and handles rate limiting; returns:
<ul style="margin:8px 0 0 18px;line-height:1.6;color:#050505;">
<li><code>text</code> (final answer)</li>
<li><code>sources?</code> (optional citations/links)</li>
</ul>
</td>
</tr>
</table>

<div style="margin-top:10px;padding:12px;border:3px solid #050505;background:#FAFAFA;box-shadow:6px 6px 0 #050505;color:#050505;">
<strong>Why a proxy?</strong>
<ul style="margin:8px 0 0 18px;line-height:1.65;font-size:15px;color:#050505;">
<li><strong>Security</strong>: API keys stay server-side (never shipped to the client).</li>
<li><strong>Control</strong>: rate limiting + a consistent request/response shape.</li>
<li><strong>Upgrades</strong>: swap Gemini versions without rewriting the frontend.</li>
<li><strong>Freshness</strong>: when asked about “latest work”, the assistant is instructed to check my GitHub activity.</li>
</ul>
</div>
</div>
